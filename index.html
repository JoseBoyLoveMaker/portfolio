Interface e estrutura do projeto

O que esse sistema entrega

O projeto foi construído para centralizar o gerenciamento de servidores Discord em um único painel, conectando bot, API e banco de dados.

Destaques técnicos

  • Arquitetura multi-servidor com separação por guild
  • Integração entre bot Discord, API ASP.NET e painel web
  • Persistência e organização de dados com MongoDB
  • Autenticação de administrador via Discord OAuth2
  • Painel administrativo para configuração em tempo real
